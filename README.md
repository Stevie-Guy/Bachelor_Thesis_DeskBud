# Bachelor_Thesis_DeskBud
Desk Bud is an physical device, used as an AI assistant. He can help you with various tasks or give you basic information. It uses a ... variables AI model, modified with some special features.

### Desk Bud Application
Desk Bud has an application used to track the water level of the user, helping with hydratation and a healthy lifestyle. Based on the current water level you introduce, the device can suggest the user to drink more water and ask questions to make him self-conscious about the fact that he didn't drink water that day.

### Cloud features it uses:
- Google Calendar API
- Weather App API
